## TextFixed Data Type

This Data Type generates a fixed number of random words.  


### Example API Usage

```javascript
{
    "numRows": 20,
    "rows": [
        {
            "type": "TextRandom",
            "title": "text",
            "settings": {
                "startsWithLipsum": false,
                "minWords": 2,
                "maxWords": 10
            }
        }
    ],
    "export": {
        "type": "JSON",
        "settings": {
            "stripWhitespace": false,
            "dataStructureFormat": "complex"
        }
    }
}
```
