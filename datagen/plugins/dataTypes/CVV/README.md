## CVV Data Type

This Data Type generates a random CVV Number for a credit card.

### Example API Usage

```javascript
{
    "numRows": 20,
    "rows": [
        {
            "type": "CVV",
            "title": "CVV"
        }
    ],
    "export": {
        "type": "JSON",
        "settings": {
            "stripWhitespace": false,
            "dataStructureFormat": "complex"
        }
    }
}
```
